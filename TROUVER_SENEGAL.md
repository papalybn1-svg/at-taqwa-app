# 🔍 Comment Trouver le Sénégal dans la Liste

**Problème :** Le Sénégal n'apparaît pas dans la liste visible

---

## 🎯 Solution : Utiliser la Recherche

### Méthode la Plus Simple

1. **Cliquez** dans le champ de sélection de pays
2. **Tapez directement** : `Sénégal` (avec l'accent)
3. **OU tapez** : `SN` (code pays)
4. La liste se **filtre automatiquement**
5. **Sélectionnez** "Sénégal (SN)"

---

## 📍 Position du Sénégal

**Ordre alphabétique :**
- ... (pays avant)
- **Sao Tomé-et-Principe (ST)**
- **Sénégal (SN)** ← **ICI**
- **Serbie (RS)**
- ... (pays après)

**Dans votre liste visible, vous voyez :**
- "Samoa américaines (AS)"
- "Serbie (RS)" ← Le Sénégal devrait être **JUSTE AVANT** Serbie

**Le Sénégal est entre "Samoa américaines" et "Serbie" !**

---

## ✅ Action Immédiate

### Option 1 : Recherche (Recommandé)

1. **Cliquez** dans le champ de sélection
2. **Effacez** tout ce qui est écrit
3. **Tapez** : `Sénégal`
4. **Sélectionnez** "Sénégal (SN)"

### Option 2 : Défilement Vers le Haut

1. **Cliquez** dans le champ de sélection
2. **Faites défiler VERS LE HAUT** depuis "Serbie"
3. **Cherchez** "Sénégal (SN)" (juste avant "Serbie")
4. **Sélectionnez** le Sénégal

### Option 3 : Tapez le Code Pays

1. **Cliquez** dans le champ de sélection
2. **Tapez** : `SN`
3. **Sélectionnez** "Sénégal (SN)"

---

## 💡 Si Ça Ne Fonctionne Pas

### Vérifications

1. **Vérifiez l'orthographe** : "Sénégal" (avec accent sur le premier "e")
2. **Essayez sans accent** : "Senegal" (si la recherche ne fonctionne pas avec l'accent)
3. **Utilisez le code pays** : "SN"
4. **Faites défiler manuellement** entre "Samoa américaines" et "Serbie"

---
