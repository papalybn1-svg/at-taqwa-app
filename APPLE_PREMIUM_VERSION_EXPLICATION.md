# 💎 Version Premium : Comment Ça Marche avec Apple ?

## ❓ Situation

Tu ne vends pas ton app, mais tu vends une **version premium** (abonnement, déblocage de fonctionnalités, etc.).

## ✅ Réponse

**Ça dépend de COMMENT tu vends la version premium :**

### Option 1 : Version Premium via In-App Purchase (Apple) ✅

**Si tu vends la version premium DANS l'app via le système d'Apple :**
- ✅ **Apple gère TOUT** (comme pour une app payante)
- ❌ **Tu n'utilises PAS PayDunya**
- ✅ **Apple collecte l'argent**
- ✅ **Apple te paie** (moins sa commission)

**Exemples :**
- Abonnement mensuel/annuel dans l'app
- Déblocage de fonctionnalités premium
- Achat de crédits/points dans l'app
- Achat de contenu premium dans l'app

### Option 2 : Version Premium via Site Web (PayDunya) ⚠️

**Si tu vends la version premium SUR TON SITE WEB :**
- ✅ **Tu peux utiliser PayDunya**
- ⚠️ **MAIS avec restrictions Apple**
- ❌ **Tu ne peux PAS promouvoir dans l'app**
- ⚠️ **Risque de rejet par Apple**

**Règles Apple :**
- Tu ne peux pas rediriger les utilisateurs vers ton site web pour payer
- Tu ne peux pas promouvoir les prix/offres de ton site web dans l'app
- Tu ne peux pas mentionner que c'est moins cher sur ton site web

---

## 📱 Option 1 : Version Premium via In-App Purchase (Recommandé)

### Comment Ça Marche

```
1. Utilisateur ouvre ton app (gratuite)
   ↓
2. Utilisateur voit "Passer à Premium"
   ↓
3. Utilisateur clique → Apple demande le paiement
   ↓
4. Apple collecte l'argent (via compte Apple/iTunes)
   ↓
5. Apple prend sa commission (30% ou 15%)
   ↓
6. Apple te paie le reste (70% ou 85%)
   ↓
7. Tu débloques les fonctionnalités premium
```

### Avantages

✅ **Simple** : Apple gère tout  
✅ **Sécurisé** : Pas de gestion de sécurité  
✅ **Conforme** : Respecte les règles Apple  
✅ **Automatique** : Pas de gestion manuelle  
✅ **Intégré** : Expérience utilisateur fluide  

### Commission Apple

**Pour les abonnements :**
- **30%** la première année
- **15%** après 1 an de service continu

**Pour les achats uniques :**
- **30%** (ou 15% si programme petites entreprises)

### Exemple Concret

**Scénario :** Abonnement Premium à 10€/mois

**Mois 1-12 :**
- Utilisateur paie : 10€
- Apple prend : 3€ (30%)
- Tu reçois : 7€

**Mois 13+ :**
- Utilisateur paie : 10€
- Apple prend : 1,50€ (15%)
- Tu reçois : 8,50€

---

## 🌐 Option 2 : Version Premium via Site Web (PayDunya)

### Comment Ça Marche

**Tu NE PEUX PAS :**
- ❌ Rediriger les utilisateurs vers ton site web depuis l'app
- ❌ Promouvoir les prix/offres de ton site web dans l'app
- ❌ Mentionner que c'est moins cher sur ton site web
- ❌ Créer un lien direct vers la page de paiement

**Tu PEUX :**
- ✅ Vendre la version premium sur ton site web
- ✅ Utiliser PayDunya pour collecter les paiements
- ✅ Mentionner que la version premium existe (sans prix)
- ✅ Laisser les utilisateurs trouver ton site web par eux-mêmes

### Risques

⚠️ **Apple peut rejeter ton app** si tu violes les règles  
⚠️ **Apple peut retirer ton app** si tu violes les règles  
⚠️ **Expérience utilisateur moins fluide** (sortir de l'app)
