## 🔍 Comment Trouver le Sénégal

### Le Sénégal EST dans la Liste

**Le Sénégal est listé sous le nom :** **"Sénégal"** avec le code pays **"SN"**

### Comment le Trouver

**Méthode 1 : Faire Défiler la Liste**
- La liste est **longue** (plus de 200 pays)
- Le Sénégal se trouve **après** les pays commençant par "S"
- Faites défiler jusqu'à la lettre **"S"**
- Cherchez **"Sénégal (SN)"**

**Méthode 2 : Utiliser la Recherche**
- Cliquez dans le champ de sélection de pays
- Tapez **"Sénégal"** ou **"SN"**
- La liste se filtrera automatiquement
- Sélectionnez **"Sénégal (SN)"**

**Méthode 3 : Ordre Alphabétique**
